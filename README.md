### NLTK Data Installation
To download the corpora used by nltk, run the Python interpreter and type the commands:
```bash
import nltk
nltk.download()
```
Choose which corpora and where to save them on your system


## Training

```python
python training.py
```

## Launching the Chatbot

```python
python chatbot.py
```
